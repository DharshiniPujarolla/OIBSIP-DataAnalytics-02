# Data Analytics – Task 02: Exploratory Data Analysis (EDA) on Superstore Sales Dataset

This project was completed as **Task 2** of the **Oasis Infobyte Summer Internship Program (OIBSIP) – Data Analytics track**. It performs an end-to-end **Exploratory Data Analysis (EDA)** on the **Sample Superstore** retail dataset to uncover trends and insights in sales, profit, and shipping performance.

## 📌 Project Overview

The goal of this task is to analyze retail transaction data and identify the key factors driving sales and profitability. The notebook walks through data loading, cleaning, and exploration, and generates a series of visualizations to summarize the findings — saving them as image files alongside a cleaned/EDA-ready CSV.

Typical steps covered in the notebook include:

- Loading and inspecting the raw Superstore dataset
- Cleaning and preparing the data for analysis
- Analyzing sales and profit by **Category**, **Sub-Category**, **Segment**, and **Region**
- Examining the relationship between **Discount** and **Profit**
- Studying **monthly and yearly sales trends**
- Reviewing **shipping mode** performance
- Computing a **correlation heatmap** across numeric features
- Exporting cleaned data and all charts generated during the analysis

## 📊 Dataset

**Sample - Superstore.csv** is a widely used retail dataset containing order-level information such as:

| Column | Description |
|---|---|
| `Order ID`, `Order Date`, `Ship Date` | Order and shipping details |
| `Ship Mode` | Shipping method used |
| `Customer ID`, `Customer Name`, `Segment` | Customer information |
| `Region`, `State`, `City` | Geographic details |
| `Category`, `Sub-Category`, `Product Name` | Product details |
| `Sales` | Revenue generated by the order |
| `Quantity` | Number of units ordered |
| `Discount` | Discount applied on the order |
| `Profit` | Profit earned on the order |

## 🛠️ Tools & Libraries Used

- **Python 3**

## 🚀 Getting Started

### Prerequisites
Make sure you have Python 3 installed along with the required libraries:

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

### Running the Notebook

1. Clone this repository:
   ```bash
   git clone https://github.com/DharshiniPujarolla/OIBSIP-DataAnalytics-02.git
   cd OIBSIP-DataAnalytics-02
   ```
2. Launch Jupyter Notebook:
   ```bash
   jupyter notebook main.ipynb
   ```
3. Run the cells sequentially to reproduce the analysis and regenerate the charts.

## 📈 Key Insights

The notebook explores questions such as:

- Which product categories and sub-categories generate the most sales and profit?
- Are there categories with high sales but low (or negative) profit?
- How does discount level impact profit margins?
- Which regions and customer segments perform best?
- What seasonal or year-over-year trends exist in sales?
- How does shipping mode relate to order performance?
- Which numeric features are most strongly correlated (e.g., Sales, Profit, Discount, Quantity)?

*(Refer to `main.ipynb` and the generated `.png` charts for the detailed visualizations and conclusions.)*
